## 📥 Instalação

Clone o repositório:

```bash
git clone https://github.com/Liiviy/BOT.git
```

Entre na pasta:

```bash
cd BOT
```

Instale as dependências:

```bash
npm install
```

Inicie o bot:

```bash
node .
```

---

## ⚙️ Configuração

Edite os arquivos de configuração conforme a documentação do GitBook.

Exemplo:

```env
TOKEN=SEU_TOKEN
CLIENT_ID=SEU_CLIENT_ID
GUILD_ID=SEU_GUILD_ID
```
